# PORTAFOLIO

## Portafolio Personal hecho en REACT 


***Como Empezar*** 

>Clonar el Repo
`git clone https://github.com/AngelVelay/PORTAFOLIO_PERSONAL/`

> Instalar las dependencias necesarias 
`npm install `

> Ejecuta el comando para ver la app 
`npm start`
